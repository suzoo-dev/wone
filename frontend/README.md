# The Frontend

Welcome to **The Frontend**! This part of the project is built using React, TypeScript, and Vite. Below you'll find instructions for setting up and running the project.

## Requirements

- Node.js (v22)
- npm or yarn

## Getting Started

1. **Install Dependencies**:

```bash
npm install
# or
yarn install
```

2. **Run the Application**:
   Start the application with:

```bash
npm run dev
# or
yarn dev
```

3. **Ensure you have the backend running**:

- [Backend README](../backend/README.md)
